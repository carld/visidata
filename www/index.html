VISIDATA

Brand New Tooling. Same Great Shell.

VisiData gives you a powerful way to manage your data from the comfort of your terminal.

VisiData is free, open source, and can be installed in seconds. Just open your terminal and enter:

pip3 install visidata

Easy/Convenient/Powerful/Intuitive.

Every command is only one or two keystrokes.

Universal.

Immediately start browsing terabyte .csv files, sqlite databases, JSON APIs, and any other source that can be presented as tabular data.

Extensible.

Create your own commands, data manipulations, and terminal applications, with a minimum of code and a maximum of flexibility.

Ready to get started?

Start with vdtutor.

Learn the basics of VisiData with the builtin vdtutor, which guides you through an interactive tutorial.
